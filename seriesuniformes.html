Series Uniformes (Anualidades)

Las series uniformes son sucesiones de pagos iguales realizados a intervalos de tiempo iguales a una tasa de interés compuesta. Se utilizan comúnmente en créditos comerciales y bancarios.

Tipos de Anulidades

  • Anualidad Vencida: Los pagos se realizan al final de cada período.
  • Anualidad AnticipadaLos pagos se realizan al inicio de cada período.
  • Anualidad Diferida: El primer pago se realiza unos periodos después de iniciada la operación financiera.
  • Anualidad Perpetua: No existe un límite de tiempo para los pagos, es decir, n tiende a infinito

Fórmulas

Valor Presente (VP) para una Anualidad Vencida:

𝑉𝑝 = 𝐴 [(1 − (1 + 𝑖)^−𝑛) / 𝑖]

Valor Futuro (VF) para una Anualidad Vencida:

𝑉𝑓 = 𝐴 [((1 + 𝑖)^𝑛 − 1) / 𝑖]

Ejemplo

Calcular el valor presente de una anualidad vencida con pagos mensuales de $2,000,000 a una tasa de interés del 24% anual durante 36 meses.

𝑉𝑝 = 2,000,000 [(1 − (1 + 0.02)^−36) / 0.02] = $50,977,685
